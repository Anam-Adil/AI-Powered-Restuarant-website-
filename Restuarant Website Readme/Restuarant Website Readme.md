## Project Title

AI-Powered Restaurant Website

 Project Description
This project is a modern, responsive restaurant website developed as part of Assignment 01. The website provides an engaging user experience with multiple sections including Home, Explore Menu, Best Sellers, About, and Contact. It also includes functional features such as Add to Cart and Pickup options, making it a complete front-end solution for a restaurant business.

## AI Tool Used

Lovable AI was used to generate and design the website layout, structure, and visual elements efficiently.

 Prompt Used
"Create a modern restaurant website with sections including Home, Explore Menu, Best Sellers, About, and Contact. Add functional buttons such as Add to Cart and Pickup. Include food items like burgers, fries, zinger rolls, zinger paratha, cold drinks, and salads. The design should be visually appealing and user-friendly."

## Features

    •Responsive and modern UI design

    •Home page with attractive layout

    •Explore Menu with categorized items

    •Best Sellers section

    •Add to Cart functionality

    •Pickup option integration

    •About and Contact pages

    •Food images (burgers, fries, zinger rolls, etc.)
